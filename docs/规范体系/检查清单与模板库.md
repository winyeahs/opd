# 检查清单与模板库

**文档版本**：V1.0  
**发布日期**：2026-02-17  
**文档性质**：规范性文档  
**所属体系**：一人AI开发工程规范体系

---

## 一、总则

### 1.1 文档目的

本手册汇集一人AI开发工程中各阶段、各角色的检查清单和标准模板，为项目开发提供即用型工具，确保工作规范、完整、高效。

### 1.2 使用说明

```
┌─────────────────────────────────────────────────────────────┐
│                    检查清单使用说明                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 在相应阶段开始前，查阅对应的检查清单                      │
│  2. 按照检查清单逐项执行和确认                                │
│  3. 使用模板生成标准化的交付物                                │
│  4. 所有检查项完成后才能进入下一阶段                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
